# Polymarket CLOB API Documentation

## Base Information

- **REST Host:** https://clob.polymarket.com  
- **WSS Host:** wss://ws-subscriptions-clob.polymarket.com

---

## 1. REST API

### GET /book
Retrieves the order book summary for a specific token.

**Query Parameters**
- `token_id` (string, required): The unique identifier for the token (Asset ID).

**Response Schema (200 OK)**

```json
{
  "market": "string",
  "asset_id": "string",
  "timestamp": "string",
  "hash": "string",
  "bids": [
    { "price": "string", "size": "string" }
  ],
  "asks": [
    { "price": "string", "size": "string" }
  ],
  "min_order_size": "string",
  "tick_size": "string",
  "neg_risk": "boolean"
}
```

## 2. WebSocket (WSS) Overview

### Connection & Authentication
- **User Channel:** Requires authentication  
- **Market Channel:** Public

**Subscription Message**
```json
{
  "type": "string",
  "assets_ids": ["string"],
  "markets": ["string"],
  "auth": {
    "apiKey": "string",
    "secret": "string",
    "passphrase": "string"
  }
}
```

**Dynamic Subscription (Market Channel)**
```json
{
  "assets_ids": ["string"],
  "operation": "subscribe"
}
```
